# Azure Powershell Automations

This repository contains various Powershell scripts used to automate reporting tasks related to Azure Entra ID and Microsoft 365. 

# Scripts included
**distribution-list-analysis.ps1**

Gathers records on every distribution list member, with the distribution list they are assigned to.

**shared-mailbox-analysis.ps1**

Gathers data on all shared mailboxes and their delegates. 

# Use of AI

For some scripts, identified accordingly, AI was used to generate the code. In these cases, I overviewed the steps taken and would make changes where appropriate.
